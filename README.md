# Homebrew Tap: tankibaj/tap

Stable Homebrew formulas for `afm-api`.

## Install

```bash
brew tap tankibaj/tap
brew install afm-api
```

## Upgrade

```bash
brew update
brew upgrade afm-api
```

## Versioned formulas

This tap also keeps versioned formulas for major/minor tracks:

- `afm-api@1.0`
- `afm-api@1.1`

Install a pinned track:

```bash
brew install afm-api@1.1
```

## Verify

```bash
brew info afm-api
which afm-api
afm-api --version
```

## Prerelease channel

RC builds are published in a separate tap:

- `tankibaj/prerelease` ([homebrew-tap-prerelease](https://github.com/tankibaj/homebrew-tap-prerelease))
